Predicting X-Ray Component Failure by Utilizing Machine Learning Techniques
This repository contains the source code and notebooks developed as part of a final project in collaboration with JBT Marel ehf., carried out at Reykjavik University, Department of Engineering, spring 2026.
Project Overview
The project investigates the use of machine learning to predict component failure in X-ray generators used in JBT Marel's SensorX machines. Telemetry data collected from the machines was used to develop and evaluate three supervised learning models: Random Forest (RF), Gradient Boosted Trees (GBTs), and Multilayer Perceptron (MLP). The MLP model was selected for further development and delivered to JBT Marel as a practical predictive maintenance tool.

Requirements
All notebooks were developed and run within the Databricks environment provided by JBT Marel, using Apache Spark and the PySpark MLlib library.
